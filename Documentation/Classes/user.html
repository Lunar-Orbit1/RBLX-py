Methods

Method: sendMessage

subject: str body: str
returns: bool

You need to be authenticated to run this

sendMessage sends one of the older "email" style message to the user from the currently authenticated account.

subject: The subject of the message
body: The body content of the message

import rblx-py as rblx
client = rblx.Client(your_token)
user = rblx.getUser(userid)
user.sendMessage("Hello there!", "hello world!")

Method: get_avatar_headshot

size: ThumbnailSize=ThumbnailSize.420x420 format: ImageFormat=ImageFormat.png isCircular: bool=False

Fetches the user's avatar headshot. Returns a url which points to the image

size: The ThumbnailSize of the image (default ThumbnailSize.420x420)
format: The format of the image (default ImageFormat.png)
isCircular: Is the returned image round? (default False)
